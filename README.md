# Snake Game

A classic browser-based Snake Game built with **HTML**, **CSS**, and **JavaScript** using the **HTML5 Canvas API**. This project demonstrates core front-end development concepts such as keyboard interaction, game-state management, collision detection, score tracking, and dynamic rendering on a grid-based board.

## Features

- Keyboard-controlled snake movement
- Real-time score updates
- Random food generation
- Snake growth after collecting food
- Edge wrap-around functionality
- Self-collision detection with game-over reset
- Clean and minimal dark-themed interface

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Canvas API**

## Project Preview

The game starts with a snake placed on the board. The player controls its movement using the arrow keys and collects food to increase the score. Each time food is eaten, the snake grows longer, making movement more challenging. The game ends when the snake collides with its own body.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/harshitha1106/snake-game.git
```

### Open the Project

```bash
cd snake-game
```

### Run the Project

Open the `index.html` file in your browser.

You can simply double-click the file, or if you use **VS Code**, open the folder and run it with **Live Server** for a better development experience.

## How to Play

- Use the **Arrow Keys** to move the snake
- Eat the **red food** to increase your score
- Avoid colliding with the snake’s own body
- The game restarts after a game-over condition
